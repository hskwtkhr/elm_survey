## ダッシュボードへのアクセス

### ログイン方法

1. ブラウザで以下のURLにアクセス：
   ```
   https://elm-survey.vercel.app/login
   ```
   または
   ```
   https://elm-survey.vercel.app/dashboard
   ```
   （未ログインの場合は自動的にログインページにリダイレクトされます）

2. ログイン情報を入力して「ログイン」ボタンをクリック

3. ログイン成功後、ダッシュボードページが表示されます

---

## ダッシュボードの機能

### 1. 統計情報

ダッシュボードの上部に、以下の統計情報が表示されます：

- **総回答数**: これまでに収集されたアンケートの総数
- **大変満足**: 「大変満足」と回答した人数
- **満足**: 「満足」と回答した人数
- **Google Review Button Clicks**: 口コミボタンがクリックされた回数（「詳細」ボタンで院ごとの内訳を確認可能）

### 2. フィルタリング機能

**院ごとの絞り込み**
- 「院を選択」ドロップダウンから、特定の院を選択
- 「すべて」を選択すると、全院のデータが表示されます

**日付指定**
- 「開始日」と「終了日」を指定して、期間を絞り込めます
- カレンダーアイコンをクリックして日付を選択
- 日付を指定しない場合は、全期間のデータが表示されます

**リセット**
- 「リセット」ボタンをクリックすると、すべてのフィルタが解除されます

### 3. グラフ表示

以下の4つのグラフが表示されます：

1. **満足度グラフ**: 満足度別の回答数の円グラフ
2. **施術メニューグラフ**: 施術メニュー別の回答数の円グラフ
3. **年齢層グラフ**: 年齢層別の回答数の円グラフ
4. **院別グラフ**: 院別の回答数の円グラフ

各グラフは、フィルタリング条件に応じて自動的に更新されます。

### 4. データテーブル

アンケート結果の一覧が表形式で表示されます。

**表示項目**:
- 記入日時
- 院名
- 先生名
- 施術日
- 施術メニュー
- 性別
- 年齢層
- 満足度
- 操作（編集・削除ボタン）

**ページネーション**:
- 1ページあたり50件表示
- ページ下部の「前へ」「次へ」ボタンでページを移動
- 現在のページ番号と総ページ数が表示されます

## アンケート結果の確認

### 基本的な確認方法

1. ダッシュボードにログイン
2. 統計情報で全体の傾向を把握
3. グラフで視覚的にデータを確認
4. データテーブルで個別の回答を確認

### 特定の院のデータを確認する

1. 「院を選択」ドロップダウンから、確認したい院を選択
2. 統計情報、グラフ、データテーブルが自動的に更新されます

### 特定の期間のデータを確認する

1. 「開始日」と「終了日」を指定
2. 指定した期間内のデータのみが表示されます

## アンケート結果の編集・削除

### 編集方法

1. データテーブルで、編集したいアンケート結果の行の「編集」ボタンをクリック
2. 編集モーダルが開きます
3. 以下の項目を編集できます：
   - 院名
   - 先生名
   - 施術日
   - 施術メニュー
   - 性別
   - 年齢層
   - 満足度
4. 編集内容を確認して「保存」ボタンをクリック
5. 「キャンセル」ボタンで編集をキャンセルできます

### 削除方法

1. データテーブルで、削除したいアンケート結果の行の「削除」ボタンをクリック
2. 確認ダイアログが表示されます
3. 「OK」をクリックすると削除されます
4. **注意**: 削除したデータは復元できません

## データのエクスポート

### CSVエクスポート

1. ダッシュボード上部の「CSVエクスポート」ボタンをクリック
2. 現在のフィルタリング条件に基づいて、CSVファイルがダウンロードされます
3. ファイル名は `survey_export_YYYY-MM-DD.csv` の形式です

**エクスポートされるデータ**:
- 記入日時
- 院名
- 先生名
- 施術日
- 施術メニュー
- 性別
- 年齢層
- 満足度
- 結果への満足度
- カウンセリングへの満足度
- 院内の雰囲気
- スタッフの対応
- 伝えたいこと（メッセージ）

**フィルタリングとの連携**:
- 院を選択している場合、その院のデータのみエクスポート
- 日付を指定している場合、その期間のデータのみエクスポート
- フィルタをリセットしてからエクスポートすると、全データがエクスポートされます

## アンケート項目の編集

ダッシュボード右上の「設問の編集」ボタンから、アンケートの各項目を自由にカスタマイズできます。

### 1. 設問選択肢の編集
以下の項目の選択肢を追加・編集・削除・並べ替えできます：
- 性別
- 年齢層
- 満足度（各種）

### 2. 質問文の編集
アンケートフォームに表示される質問のタイトル（例：「第1問」など）を編集できます。

### 3. 施術メニューの編集
「施術メニュー」の選択肢を追加・編集・削除・並べ替えできます。

### 4. 先生名の編集
各院に所属する先生の名前を追加・編集・削除・並べ替えできます。
（院を選択してから、その院の先生を編集します）

### 注意事項
- **院の追加・削除**: 現在、管理画面からは行えません。開発者によるデータベース操作が必要です。

## よくある質問（FAQ）

### Q: ログインできない
A: 以下を確認してください：
- ユーザー名とパスワードが正しいか
- ブラウザのキャッシュをクリアして再試行
- シークレットモードで試す

### Q: データが表示されない
A: 以下を確認してください：
- フィルタリング条件が正しいか（特に日付の範囲）
- 「リセット」ボタンをクリックしてフィルタを解除
- ブラウザをリロード

### Q: CSVエクスポートができない
A: 以下を確認してください：
- ブラウザのポップアップブロックが有効になっていないか
- ダウンロードフォルダを確認
- ブラウザをリロードして再試行

### Q: 院を追加したい
A: 現在、ダッシュボードから院を追加する機能はありません。開発者に依頼して、データベースに直接追加する必要があります。
